Class : AjaxDropdown

AjaxDropdown class populates the data fetched from database into a sub category dropdown on change event of category dropdown. The configuration settings can be changed as per need of application. It makes use of XML as transitional language between server side language and client end script. The component is multibrowser, cross database, multi server side language.

For Installaton ..
	
	1. Run db.sql file on your local or remote database..
	2. Change the configuration settings for host, database name, username and password  inside getagents.php file.
	3. Run index.php file using php compatible web server.
